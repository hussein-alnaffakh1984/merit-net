# Reproduction steps (seed = 42)

1. `conda env create -f environment.yml && conda activate merit-net`
2. `python preprocessing.py --raw "Dataset - Updated.csv" --out mendeley_cleaned.csv`   (1,205 -> 1,164)
3. `python make_splits.py --data mendeley_cleaned.csv`   (writes cv_fold_indices.json, holdout_indices.json)
4. `python merit_net.py`   (trains the five-model ensemble; Tables 2-5, confusion matrices, Trust Score)
5. `python baseline_configs.py`   (Table 3 baselines, leakage-free)

Hyperparameters: see config.json (exact ensemble settings, conformal split 582/582, Trust Score weights).

Files generated by the run (committed alongside): fold-level predictions, the deduplicated
sensitivity-analysis outputs, and the per-table/per-figure result CSVs.
